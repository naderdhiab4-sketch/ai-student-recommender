# Flutter app - README

المجلد flutter_app يحتوي scaffold لتطبيق Flutter مع لغات عربية.

متطلبات:
- Flutter SDK
- Android SDK أو بيئة تشغيل سطح مكتب (Windows/Mac)

تشغيل:
1. افتح المجلد flutter_app في VS Code أو Android Studio
2. flutter pub get
3. flutter run

قنوات API الافتراضية تشير إلى http://localhost:8000 — يمكن تعديلها في ملف config داخل المشروع.
